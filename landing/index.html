TDEN
可信数字生态网络
Trusted Digital Ecosystem Network

让通讯更自由,但有边界。
一个由各国主权身份节点(SIN)组成的去中心化身份基础设施, 把"用户主权"具体化成可在浏览器里独立验签的访问回执。

A decentralized identity infrastructure operated as a federation of national-level Sovereign Identity Nodes (SINs). Every third-party data query produces an Ed25519-signed receipt the user can verify in their own browser ─ no need to trust the gateway's word.

🔁 系统流程图 System architecture & data flow

读法 / How to read: 上层是客户端(用户的手机/电脑/桌面端 app + 第三方 RP), 中层是 tden-gateway(单一对外 HTTPS 入口,把 OIDC、身份、场景包、回执、Webhook 集中处理), 下层左边是专用后端服务(人脸识别、联邦学习聚合、众包计算、AI 助手、本地身份金库), 最下层是 tden-chain(Cosmos SDK 区块链,把关键操作锚定上链;多 SIN 间靠链上 federation 模块互联)。

How it flows: Clients (mobile / web / desktop / 3rd-party RPs) only ever talk to one HTTPS host: the gateway. The gateway dispatches to specialised backends (face service, FL server, compute pool, miner AI) and anchors critical state on the chain. Sovereign Identity Nodes (CN/US/JP) are independent chains linked by explicit on-chain trust edges in the federation module.
